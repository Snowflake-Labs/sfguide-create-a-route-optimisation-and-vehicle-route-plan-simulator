## Creating Custom Industry-Specific Routing Examples

Follow this workflow to build new industry scenarios using `nyc_beauty_routing_simple.py` as a template.

### Design
- Select industry and define realistic constraints
- Design data structures (depots, fleet, jobs)

### Data Creation (Internal Marketplace)
- Create tables in `FLEET_MANAGEMENT.PUBLIC`
- Include optimization arrays and `GEOGRAPHY` columns

### Listing and Views
- Create share and organizational listing
- In lab instance, request listing and create views for attendees

### Streamlit App
- Clone the template app and adjust branding, data sources, and parameters

### Testing & Documentation
- Validate data and optimization results
- Update docs and navigation in MkDocs
