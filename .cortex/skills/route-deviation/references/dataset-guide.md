# Dataset Guide

## Data Source

All data is sourced from `SYNTHETIC_DATASETS.UNIFIED`, generated by Data Studio. The CONFIG table controls which vehicle type and region the ETL processes.

## Source Tables (SYNTHETIC_DATASETS.UNIFIED)

| Table | Description |
|-------|-------------|
| FACT_VEHICLE_TELEMETRY | GPS telemetry with speed, heading, status, detour flags |
| FACT_TRIPS | Trip records with actual and planned routes, distances, detour metrics |
| DIM_FLEET | Vehicle/driver metadata: type, profile, home location |
| DIM_POIS | Points of interest: warehouses, stores, rest stops |
| DIM_TRIP_SCHEDULE | Trip schedule with OD pairs and shift assignments |

## Projection Views

Views in `FLEET_INTELLIGENCE.ROUTE_DEVIATION` filter UNIFIED tables by CONFIG and rename columns:

| View | Source | Key Transforms |
|------|--------|---------------|
| VW_TRUCK_TELEMETRY | FACT_VEHICLE_TELEMETRY | VEHICLE_ID -> TRUCK_ID |
| VW_TRIP_DEVIATION | FACT_TRIPS | ROUTE_GEOG -> ACTUAL_PATH, PLANNED_ROUTE_GEOG -> EXPECTED_PATH |
| VW_FLEET | DIM_FLEET | VEHICLE_ID -> TRUCK_ID, OPERATING_MODE -> TRUCK_TYPE |
| VW_TRIP_SCHEDULE | DIM_TRIP_SCHEDULE | VEHICLE_ID -> TRUCK_ID, STATUS -> ROUTE_VARIATION |
| VW_POIS | DIM_POIS | LOCATION_ID -> ID |

## ETL Output Tables

| Table | Description |
|-------|-------------|
| TRIP_DEVIATION_ANALYSIS | Actual vs expected comparison with deviation flags per trip |
| DRIVER_DEVIATION_SUMMARY | Per-driver aggregated deviation statistics |
| DAILY_DEVIATION_TRENDS | Daily aggregated deviation metrics |

## Deviation Classification

Trips are classified based on distance deviation percentage:
- **On Route**: deviation < 20%
- **Deviated**: deviation >= 20% or IS_DETOUR flag from Data Studio

## Vehicle Types

Data Studio supports multiple vehicle types. Update CONFIG to switch:

| Vehicle Type | ORS Profile | Typical Use |
|-------------|-------------|-------------|
| hgv | driving-hgv | Trucks, logistics |
| car | driving-car | Taxis, ride-sharing |
| ebike | cycling-electric | E-bike couriers, food delivery |
