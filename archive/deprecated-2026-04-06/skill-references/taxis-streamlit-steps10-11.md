### Upload Files

```sql
PUT 'file://assets/streamlit/Taxi_Control_Center.py' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/extra.css' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/logo.svg' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/environment.yml' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/city_config.py' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/pages/1_Driver_Routes.py' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/pages/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
PUT 'file://assets/streamlit/pages/2_Heat_Map.py' @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/pages/ AUTO_COMPRESS=FALSE OVERWRITE=TRUE;
```

### Verify Upload

```sql
LIST @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi/;
```

### Create Streamlit App

```sql
CREATE OR REPLACE STREAMLIT FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.TAXI_CONTROL_CENTER
  FROM @FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.STREAMLIT_STAGE/taxi
  MAIN_FILE = 'Taxi_Control_Center.py'
  QUERY_WAREHOUSE = ROUTING_ANALYTICS
  TITLE =  'Taxi Control Center'
  COMMENT = '{"origin":"sf_sit-is-fleet", "name":"oss-fleet-intelligence-taxis", "version":{"major":1, "minor":0}, "attributes":{"is_quickstart":1, "source":"streamlit"}}';
```

### Set Live Version

```sql
ALTER STREAMLIT FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.TAXI_CONTROL_CENTER ADD LIVE VERSION FROM LAST;
```

### Verify Deployment

```sql
SHOW STREAMLITS IN SCHEMA FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS;
```

### Get App URL

```sql
SELECT CONCAT('https://app.snowflake.com/', CURRENT_ORGANIZATION_NAME(), '/', CURRENT_ACCOUNT_NAME(), '/#/streamlit-apps/FLEET_INTELLIGENCE.FLEET_INTELLIGENCE_TAXIS.TAXI_CONTROL_CENTER') AS STREAMLIT_URL;
```
