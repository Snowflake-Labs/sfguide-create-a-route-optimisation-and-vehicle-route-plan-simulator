## Step-By-Step Guide

1. Deploy app:
- go to working directory of the project
- in CoCo CLI type: "use the local skill from skills/deploy-route-optimizer".
This will deploy the app in roughly 5 minutes

2. Select a custom map 
- after the app is deployed, type in CoCo CLI: "use the local skill from skills/ors-map-customization to change the map to New York". We recommend using the smallest map possible for your use case

3. (optional) deploy demo notebook and streamlit 
- prerequisite: New York map installed
- type in CoCo CLI "use the local skill from skills/deploy-demo
