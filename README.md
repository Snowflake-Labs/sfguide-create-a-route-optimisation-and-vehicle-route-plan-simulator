## Step-By-Step Guide

1. Deploy app:
- go to working directory of the project
- in CoCo CLI type: "use the local skill from skills/deploy-route-optimizer".
This will deploy the app in roughly 5 minutes

2. Customize app in case of need (select map of any other region - for MVP just European countries are supported)
- after the app is deployed, type in CoCo CLI: "use the local skill from skills/ors-map-customization to change the map to Albania"
